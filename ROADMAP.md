# 升級 Roadmap — 從課程作業到作品集

每個專案分「原始作業 → 已做的升級 → 深造方向(深度) / 延伸方向(廣度)」。
建議依你目前準備國考的時間，一次專注推進一個，優先序見最後。

## 01 央行匯率分析與預測 (旗艦)
- **原始**：一堆散落的爬蟲與 csv，資料躺著沒被用。
- **已升級**：解析 178 個央行資料集、萃取 9 幣別每日匯率、EDA(趨勢/波動)、
  次日預測(Ridge/GBR) 並與隨機漫步基準對照、誠實結論。
- **深度**：加入 GARCH 做波動度預測(方向難、波動可預測)；改做週報酬；加入交易成本的回測。
- **廣度**：把另外 170+ 央行總經序列(利率、貨幣供給、外匯存底)當特徵，做總經連動分析——
  這條最貼近你的「AI 金融投資」目標，也最能寫進留學 SOP。

## 02 影像去模糊
- **原始**：把影像做「模糊 vs 清晰」二元分類(框架有誤，且程式有 bug，已修)。
- **已升級**：重新定義為「影像還原」；提供傳統基準(Unsharp / Richardson-Lucy) + U-Net 訓練骨架。
- **深度**：訓練 U-Net / DeblurGAN，用 PSNR/SSIM 評估，跟傳統法比較。
- **廣度**：延伸到去雜訊、超解析度(super-resolution)。

## 03 水果好壞影像分類
- **原始**：good/bad banana 影像資料集，未見完整分類器。
- **已升級**：MobileNetV2 遷移學習骨架，含資料增強、早停、混淆矩陣。
- **深度**：Grad-CAM 可解釋性(模型看哪裡判斷腐壞)、擴充到多種水果/多級新鮮度。
- **廣度**：部署成 Flask/Gradio 網頁 Demo(可接你既有的 Flask 練習)。

## 04 語音訊號分析
- **原始**：MATLAB 的 ZCR/能量/f0/PCA-ICA 腳本。
- **已升級**：ZCR/能量/頻譜的 Python 移植，含語音/靜音偵測(VAD)。
- **深度**：加 MFCC 特徵 + 簡單分類器做關鍵詞辨識(你有 jarvus.wav 等素材)。
- **廣度**：把 PCA/ICA 盲訊號分離(cocktail party)也移植過來成一個獨立 Demo。

## 05 自動駕駛場景模擬
- **原始**：RoadRunner 建道路(OpenDRIVE/.xodr) + Unity 場景 + 三份簡報。
- **已升級**：整理成技術文件，說明 pipeline 與素材。
- **深度**：在 Unity 中放一台車 + 相機，接 02 的視覺模型做車道/號誌辨識。
- **廣度**：用 CARLA 之類開源模擬器重做，程式化生成場景。

## 建議優先序
1. **01 加總經特徵**(對留學＋金融投資 CP 值最高)
2. **04 加 MFCC 關鍵詞辨識**(範圍小、易完成、成就感高)
3. **03 部署成網頁 Demo**(整合你的 Flask，作品集最吸睛)
4. 02 U-Net 訓練(需要 GPU) / 05 進階(工程量大)
