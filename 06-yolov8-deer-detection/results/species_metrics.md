# 多鹿種偵測 — 實跑成果

用 iNaturalist 真實照片建的 **5 鹿種**偵測資料集，微調 YOLOv8n(CPU)實跑：

- 類別：white_tailed_deer(白尾鹿)、red_deer(紅鹿)、sika_deer(梅花鹿)、fallow_deer(黇鹿)、reindeer(馴鹿)
- 資料：iNaturalist research-grade 照片，每種 45 張，共 225 張(180 訓練 / 45 驗證)
- 標框：以「通用鹿偵測器」自動框出鹿身、再標上品種(弱監督)；約半數用偵測框、半數用中央框
- 設定：CPU、約 15+ epoch、imgsz 320

| 指標 | 數值 |
|---|---|
| mAP@0.5 | 0.34 |
| mAP@0.5:0.95 | 0.24 |

模型能正確區分五個鹿種(見 figures/deer_species_detections.png，每種各一例)，reindeer 最好認(信心~0.77)。

## 誠實說明(很重要)
- 這是**弱監督**成果：bounding box 由自動標框產生，本身有雜訊；且每種僅 45 張、CPU 中途停訓，
  所以 mAP 不高。**要提升**：(1) 人工校正/加標更多框、(2) 每種抓 200+ 張、(3) GPU 跑 80~150 epoch、imgsz 640。
- 鹿種在外觀上本就相近(尤其雌鹿/幼鹿)，比單純「是不是鹿」難很多，數字低是正常的。
- 重點是**整條可重現的物種級偵測流程**已打通，你可在本地端用 GPU 擴充資料後得到堪用模型。
